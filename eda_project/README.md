# Stock Market Analysis Project

## Overview
This project analyzes the historical performnace of **Apple (AAPL)**, **Walmart (WMT)**, and the **S&P 500 (SPY)** from 2022 to 2024.

## Key FIndings
* **Resilience:** Walmart showed both stability and growth during the 2022 market downturn, showing it as a reliable defensive asset.
* **Volatility:** Apple displayed higher volatility that was related strongly with the S&P 500.
* **Conclusion:** A portfolio would benefit greatly from holding stocks similar to Walmart during market downturns and periods of economic instability.

## Technologies Used
* Python 3.13.9
* Pandas (Data Manipulation)
* Plotly Express (Interactive Visualization)
* yfinance (Data Collection)

## How to Run
1. Clone the repository.
2. Install dependecies: "pip install -r requirments.txt"
3. Open "notebooks/stack_analysis.ipynb" and run all cells.
